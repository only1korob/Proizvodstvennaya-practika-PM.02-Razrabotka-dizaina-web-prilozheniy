# MyBakery — Система управления онлайн-пекарней 🍞

Информационная система для автоматизации учета товаров и представления ассортимента.

## Описание проекта
Проект разработан в рамках модуля ПМ.03. Приложение позволяет управлять каталогом выпечки, просматривать аналитику склада и обеспечивает интерфейсы для авторизации пользователей.

## Стек технологий
* Backend: Python (Flask)
* Database: SQLite + SQLAlchemy (ORM)
* Frontend: HTML5, CSS3, Jinja2
* Оптимизация: Индексация БД, внешние статические ресурсы.

## Как запустить
1. Установите зависимости: pip install flask flask-sqlalchemy
2. Запустите сервер: python app.py
3. Откройте в браузере: http://127.0.0.1:5000

## Структура проекта
* /static/css — стили оформления (UX-оптимизация)
* /templates — HTML-шаблоны страниц
* app.py — серверная логика и модели БД
* bakery.db — локальная база данных
